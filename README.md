# wxapp-style-editor
一个系统Demo，允许录入和在线编辑小程序样式，并支持导出小程序和H5代码
# 安装运行
1、克隆项目，并执行npm install安装依赖包   
2、在目录下执行npm run dev   
3、浏览器中键入127.0.0.1:8848来访问系统   
# 特性
* 实时预览   
* sketch风格   
* 支持小程序和H5代码导出
